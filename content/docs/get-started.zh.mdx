# 开始使用

本指南将帮助你设置开发环境，并在本地运行你的应用。

## 前置条件

开始之前，请确保你已安装以下工具:

- **Node.js 18.17+**: 使用 `node -v` 检查你的版本
- **pnpm**: 软件包管理工具 (或 npm/yarn)
- **Git**: 版本控制

## 安装步骤

### 1. 克隆仓库

```bash
git clone https://github.com/your-repo/your-app.git [YOUR_APP_NAME]
cd [YOUR_APP_NAME]
```

### 2. 安装依赖

```bash
pnpm install
```

如果遇到任何问题，请尝试使用 npm:

```bash
npm install
```

### 3. 设置环境变量

重命名示例环境文件:

```bash
mv .env.example .env.local
```

然后编辑 `.env.local` 并填写你的配置值。

### 4. 设置数据库

如果你使用 Prisma，请运行迁移:

```bash
pnpm prisma migrate dev
```

### 5. 启动开发服务器

```bash
pnpm run dev
```

你的应用现在应该在 `http://localhost:3000` 上运行 🎉

## 常见问题

### 端口已被占用

如果端口3000被占用，你可以指定不同的端口:

```bash
PORT=3001 pnpm run dev
```

### 环境变量未加载

确保你的 `.env.local` 文件位于根目录并格式正确。

### 数据库连接问题

验证你的 `.env.local` 中的数据库 URL 是否正确且数据库服务器已启动。
