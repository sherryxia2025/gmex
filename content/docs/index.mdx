## Start a local server

1. In your terminal, run the following commands one-by-one:

```bash
git clone https://github.com/your-repo/your-app.git [YOUR_APP_NAME]
cd [YOUR_APP_NAME]
pnpm install
git remote remove origin
pnpm run dev
```

> This project requires Node 18.17 or greater. Type `node -v` in your terminal to check version.

2. Rename `.env.example` to `.env.local`

```bash
mv .env.example .env.local
```

3. Open `http://localhost:3000` to see your site. And voila!
