# 开始使用

嘿，创业者，欢迎 👋

这是项目的快速概览。跟着教程让你的应用运行起来。

完成后，开始这个教程，在 5 分钟内启动你的项目。让我们快速构建创业项目 ⚡️

## 启动本地服务器

1. 在终端中，逐一运行以下命令：

```bash
git clone https://github.com/your-repo/your-app.git [YOUR_APP_NAME]
cd [YOUR_APP_NAME]
pnpm install
git remote remove origin
pnpm run dev
```

> 此项目需要 Node 18.17 或更高版本。在终端中输入 `node -v` 检查版本。

2. 将 `.env.example` 重命名为 `.env.local`

```bash
mv .env.example .env.local
```

3. 打开 `http://localhost:3000` 查看你的网站。完成！

## NextJS 项目结构

* `/app` → 页面（1 个文件夹 + page.js = 1 个页面）
* `/app/api` → API 调用（1 个文件 = 1 个 API 端点）
* `/components` → React 组件
* `/lib` → 库辅助函数（认证、工具等）
* `/prisma` → 数据库模型和迁移

## 配置

应用通过环境变量进行配置。检查 `.env.local` 文件并根据需要更新值。

## .env 文件

将 `.env.example` 文件重命名为 `.env.local`。文件内容应包含应用所需的环境变量。
